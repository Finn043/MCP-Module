Lesson 0: Process la gi?

Bai hoc dau tien — Hieu Process tu con so 0

Process = Chuong trinh dang chay

Cu don gian:

  • Program = File nam tren dia cung (chua chay)
  • Process = Chuong trinh dang chay trong RAM
┌────────────────────────────────────┐
│ DIA CUNG (File System) │
│ │
│ 📄 python3 (file chua chay) │
│ 📄 chrome (file chua chay) │
│ │
│ ↓ BAT DAU CHAY ↓ │
│ │
│ ┌────────────────────────────┐ │
│ │ RAM (Memory) │ │
│ │ │ │
│ │ 🟢 Process #1 │ │
│ │ python3 dang chay │ │
│ │ │ │
│ │ 🟢 Process #2 │ │
│ │ chrome dang chay │ │
│ └────────────────────────────┘ │
└────────────────────────────────────┘

Ví du doi thuong

  • Program = Ung dung iPhone tren App Store
  • Process = Khi ban mo app do

PID la gi?

PID = Process ID — So dinh danh duy nhat cho moi process.

  • Moi process co PID rieng biet
  • Hai process khong the co cung PID

Demo 1: Xem processes dang chay

Mo Terminal va go:

Terminal
$ ps aux USER PID %CPU %MEM COMMAND user 1234 0.5 1.2 /usr/bin/python3 user 5678 2.1 8.5 /Applications/Chrome.app user 9012 0.1 0.3 /bin/bash
