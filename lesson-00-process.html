2. Mỗi Process có PID riêng

OS cấp cho mỗi process một PID (Process ID) — số định danh duy nhất.

Dùng lệnh ps aux để xem tất cả process đang chạy.

$ ps aux
USER   PID  %CPU  %MEM  COMMAND
user   123  0.5   1.2   /usr/bin/python3
user   456  2.1   8.5   /Applications/Chrome.app
user   789  0.1   0.3   /bin/bash

3. Process có bộ nhớ riêng biệt

1
Khi chạy, OS cấp phát vùng RAM riêng cho process đó
2
Các process không thể đọc RAM của nhau (mặc định)

Process bị kill → OS thu hồi RAM ngay lập tức

5. MCP — Hai process nói chuyện qua stdio

Pipe dùng stdin/stdout — chính là thứ bạn sẽ học ở Bài 01.
MCP Server = một process liên tục đọc stdin, xử lý, ghi ra stdout.

6. Spawn process trong Python

import subprocess

# Tạo process mới (spawn)
process = subprocess.Popen(
    ["python3", "mcp_server.py"],
    stdin=subprocess.PIPE,   # ← pipe stdin vào
    stdout=subprocess.PIPE,  # ← pipe stdout ra
    text=True
)

# Gửi JSON-RPC vào process
process.stdin.write('{"jsonrpc":"2.0","method":"ping","id":1}\n')
process.stdin.flush()

# Nhận response
response = process.stdout.readline()
print(response)
