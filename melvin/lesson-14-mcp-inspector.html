1. MCP Inspector la gi?

MCP Inspector la mot browser-based debugging tool duoc tich hop san trong MCP SDK. No cho phep ban test MCP server ma khong can xay dung client hoac ung dung hoan chinh.

Tai sao Inspector quan trong?

Thay vi phai viet test scripts rieng biet hoac ket noi den ung dung hoan chinh, ban co the nhanh chóng lap di lap lai qua trinh trien khai cac cong cu, kiem thu cac truong hop dac biet va gop loi trong thoi gian thuc.

Nghep cua MCP Inspector nhu mot "nha thu nghiem" cho server cua ban -- ban co the kiem tra tung linh kien (tools, resources, prompts) truoc khi lap rap chung vao ung dung chinh. Giong nhu ban test dong co truoc khi dua vao xe.

2. Khoi dong Inspector

Chi can mot cau lenh de khoi dong Inspector:

mcp dev mcp_server.py

Dieu gi xay ra khi chay lenh nay?

1
Auto-start MCP Server -- Inspector tu dong khoi dong server cua ban voi stdio transport.
2
Mo trinh duyet -- Tu dong mo trinh duyet tai http://127.0.0.1:6274.
3
Ket noi -- Inspector gui JSON-RPC messages den server cua ban thong qua stdio.

Luu y quan trong

Lenh mcp dev la mot phan cua mcp[cli] package. Dam bao ban da cai dat mcp[cli]>=1.8.0 trong dependencies.

3. Testing Tools -- Kiem thu Cong cu

Sau khi nhan Connect trong Inspector, ban chuyen den tab Tools:

Quy trinh testing Tools

1
List Tools -- Nhan nut "List Tools" de xem tat ca tools co san. Moi tool hien thi ten, mo ta, va parameter schema.
2
Select Tool -- Click vao tool ban muon test (vi du: read_doc_contents).
3
Fill Parameters -- Dien cac tham so (vi du: doc_id = "report.pdf").
4
Run Tool -- Nhan "Run Tool" de thuc thi. Xem ket qua tra ve.
Testing Tools Workflow
Connect --> List Tools --> Select Tool --> Fill Params --> Run Tool --> Result

5. Testing Prompts -- Kiem thu Prompt Templates

Chuyen den tab Prompts de test cac prompt templates:

Quy trinh testing Prompts

1
List Prompts -- Xem tat ca prompts co san voi ten va mo ta.
2
Select Prompt -- Chon prompt ban muon test (vi du: format).
3
Fill Arguments -- Dien cac tham so (vi du: doc_id = "plan.md").
4
View Messages -- Xem danh sach Message objects duoc tao ra. Day la nhung gi se gui den Claude.

6. Development Workflow -- Quy trinh Phat trien

Inspector la mot phan khong the thieu trong quy trinh phat trien MCP Server:

Development Loop
Edit Code --> Restart Inspector --> Test Tools --> Test Resources --> Test Prompts --> Fix Issues --> Repeat

Diem manh cua Inspector

1
State persistence -- Inspector duy tri trang thai server giua cac calls. Vi du: ban co the edit mot document, sau do read no de kiem tra thay doi.
2
Sequential testing -- Test nhieu tools lien tiep. Ket qua cua tool truoc anh huong den tool sau.
3
Real-time feedback -- Nhìn thấy ngay ket qua hoac loi, khong can chay toan bo ung dung.

Khi edit code, can lam gi?

Khi ban thay doi code trong mcp_server.py, ban can restart Inspector (Ctrl+C roi chay lai mcp dev mcp_server.py). Inspector se tai lai server voi code moi.
