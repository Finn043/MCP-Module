1. Bốn thành phần cốt lõi của máy tính

Mọi máy tính dù to hay nhỏ đều có 4 thành phần chính:

Thành phầnTên đầy đủVai tròVí dụ thực tế
🧠 CPUCentral Processing UnitBộ xử lý — "Não" của máy tínhChef nấu ăn
🗂️ RAMRandom Access MemoryBộ nhớ tạm — "Bàn làm việc"Mặt bàn bếp
💾 StorageHard Disk / SSDLưu trữ lâu dài — "Tủ đồ"Tủ lạnh chứa nguyên liệu
📡 I/OInput / OutputGiao tiếp với thế giới bên ngoàiCửa ra vào nhà bếp
Nhà bếp: Chef (CPU) lấy nguyên liệu từ tủ lạnh (Storage), để lên bàn bếp (RAM) rồi nấu. Khách hàng gọi món qua cửa (I/O).

2. CPU — "Não" của máy tính

CPU chỉ làm một việc duy nhất: thực hiện các lệnh (instructions) cực kỳ nhanh.

CPU thực hiện hàng tỷ lệnh mỗi giây (GHz = gigahertz).

Các lệnh đơn giản như: cộng hai số, so sánh, chuyển dữ liệu từ nơi này sang nơi khác.

CPU như chef rất nhanh tay — mỗi giây có thể thái 3 tỷ lát rau. Nhưng chef chỉ làm theo công thức, không tự sáng tạo.

Mọi thứ phải được nạp vào RAM trước, rồi CPU mới xử lý được.

4. Operating System (OS) — Người quản lý

OS (Windows, macOS, Linux) là phần mềm đặc biệt chạy đầu tiên khi bật máy. OS có nhiệm vụ:

1
Quản lý CPU — chia thời gian CPU cho nhiều chương trình
2
Quản lý RAM — cấp phát và thu hồi bộ nhớ
3
Quản lý Storage — đọc/ghi file qua File System
4
Quản lý I/O — bàn phím, màn hình, mạng, v.v.
OS như người quản lý nhà hàng — điều phối các chef (CPU), phân công bàn bếp (RAM), và kiểm soát nguyên liệu (Storage).

5. Khi bạn mở một ứng dụng — chuyện gì xảy ra?

Bạn double-click chrome
↓
OS đọc file chrome từ Storage
↓
OS nạp code vào RAM
↓
OS tạo một Process và giao cho CPU chạy
↓
Chrome hiện lên màn hình! 🎉
💡 Kết nối với bài tiếp theo:
Bước cuối — OS tạo ra một Process — chính là thứ bạn sẽ học ở Lesson 00.
Process = chương trình đang sống trong RAM, được CPU thực thi.

6. File System — Cách tổ chức dữ liệu

Storage không phải một đống dữ liệu hỗn độn. OS tổ chức nó thành File System:

/ (root)
├── Users/
│   └── baotin/
│       ├── Desktop/
│       └── Documents/
└── usr/bin/  ← các chương trình như python3, git, ...

7. Terminal — Cửa sổ giao tiếp với OS

Terminal cho phép bạn gõ lệnh trực tiếp vào OS bằng text.

Thay vì click chuột, bạn gõ: ls để xem files, cd để di chuyển thư mục, python3 app.py để chạy chương trình.
